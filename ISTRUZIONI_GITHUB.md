# Come mettere online il tuo nuovo sito Tarocchi Liquidi

Hai a disposizione la struttura completa del sito. Segui questi passaggi per aggiornare il tuo repository su GitHub:

1. **Prepara il repository**:
   - Accedi al tuo repository GitHub `tarocchiliquidi`.
   - Elimina i vecchi file (se vuoi una pulizia totale) oppure sovrascrivili.

2. **Carica i nuovi file**:
   - Trascina nel repository i seguenti elementi dalla tua cartella di lavoro:
     - `index.html` (La nuova home)
     - `traccia.html` (Il sistema QR)
     - `robots.txt` e `sitemap.xml`
     - La cartella `assets/` (Fondamentale: contiene tutte le tue foto e audio)
     - La cartella `biblioteca/`
     - La cartella `css/`
     - La cartella `js/`

3. **Verifica**:
   - Una volta completato il caricamento (Commit), attendi 1-2 minuti.
   - Visita `www.tarocchiliquidi.it` e vedrai la nuova struttura Premium attiva con tutti i tuoi contenuti reali.

4. **Nota sulle Immagini**:
   Il sito ora punta a `assets/hero_wide_clean.jpg` per la testata e `assets/logo-tl-v2.png` per il logo. Tutto è già configurato correttamente.
